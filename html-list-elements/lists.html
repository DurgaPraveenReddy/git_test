Lists

Unordered lists

The Unordered lists are created using the ul element, the list items are created using the li element. Each list item in an unordered list begins with a bullet point.

Shopping list:

  • Soap
  • Paste
  • Toothbrush
  • Coffee

The shopping list is an unordered list, which means we can buy things in whichever order we like.

Ordered lists

The ordered lists are created using the ol element, the list items are created using the li element. Each list item in an ordered list begins with a number.

My Top 5 PC Games:

  1. Witcher 3
  2. Farcry 3
  3. God Of War 2018
  4. Ghost Of Tsushima
  5. Uncharted 4

The Top 5 Games list is an ordered list, which means there is an order. In this case descending liking of the game as the list progresses.
